# Snake-Game-JavaFX
Snake game made with JavaFX.
The main goal is just to survive.

On the field, will have *dots* that could be consumed, which cause various effects, such as:
*Increase or decrease your size
*Cause positive effects on yourself, like 'speed up,' 'become invisible on others' screens,' 'leave a part of your body behind as a wall,' and more.
*Cause negative effects on opponents, like 'slow down,' 'paralyze,' 'inverted controls,' and others.

Currently, the game only supports local multiplayer using the same PC using
only the keyboard, but I'll add suport do joystick, and on the future, even
for playing online via IP.
